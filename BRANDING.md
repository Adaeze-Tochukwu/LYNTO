# Lynto Brand Colors

## Primary Palette (Sky Blue)

The primary color is a calm, trustworthy sky blue - chosen to feel professional yet non-clinical.

| Name | Hex | Usage |
|------|-----|-------|
| Primary 50 | `#f0f9ff` | Lightest backgrounds, hover states |
| Primary 100 | `#e0f2fe` | Light backgrounds |
| Primary 200 | `#bae6fd` | Borders, dividers |
| Primary 300 | `#7dd3fc` | Icons (light mode) |
| Primary 400 | `#38bdf8` | Secondary buttons |
| **Primary 500** | **`#0ea5e9`** | **Main brand color - buttons, links, accents** |
| Primary 600 | `#0284c7` | Button hover states |
| Primary 700 | `#0369a1` | Active/pressed states |
| Primary 800 | `#075985` | Dark accents |
| Primary 900 | `#0c4a6e` | Darkest shade |

## Risk Level Colors

Used for the early warning scoring system.

| Level | Main | Light Background | Usage |
|-------|------|------------------|-------|
| Green | `#22c55e` | `#dcfce7` | Score 0-2: No concerning signs |
| Amber | `#f59e0b` | `#fef3c7` | Score 3-4: Some concerning signs |
| Red | `#ef4444` | `#fee2e2` | Score 5+: Concerning signs, escalate |

## Neutral Palette (Slate)

Used for text, backgrounds, and UI elements.

| Name | Hex | Usage |
|------|-----|-------|
| Slate 50 | `#f8fafc` | Page background |
| Slate 100 | `#f1f5f9` | Card backgrounds, inputs |
| Slate 200 | `#e2e8f0` | Borders, dividers |
| Slate 300 | `#cbd5e1` | Disabled states, placeholders |
| Slate 400 | `#94a3b8` | Placeholder text, icons |
| Slate 500 | `#64748b` | Secondary text |
| Slate 600 | `#475569` | Body text |
| Slate 700 | `#334155` | Emphasized text |
| Slate 800 | `#1e293b` | Headings, primary text |
| Slate 900 | `#0f172a` | Darkest text |
